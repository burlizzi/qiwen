1.图像模块测试
    文件：不需要下载程序，只需要人工判断图像就可以。

2.IRCUT功能开启测试
    文件：需要下程程序test_ircut，再进行人工判断是否有效；

3.IRCUT功能关闭测试
    文件：需要下程程序test_ircut，再进行人工判断是否有效；

4.监听功能测试
   文件：工具这边进行下载测试程序test_monitor,再进行一直获取机器上的monitor_pcm.pcm文件，
			   获取到文件后，通过耳机进才听是否有音乐播放，循环1分钟没有获取到，那么就超时出错.

5.对讲功能测试
  文件：由PC这边传一个音频文件test_pcm.mp3，还有下载测试程序test_interphone，在机器上进行判断是否有音乐播放.

6.云台功能测试
  文件：需要下程程序test_ptz,再进行判断是否云台正确。

7.SD卡功能测试
   文件：需要下程程序test_mmc，工具后自动返回成功或失败，此时人工就可以根据此进行判断是通过还是不通过；

8.wifi功能测试
    文件：需要下程程序test_wifi，工具后自动返回成功或失败，此时人工就可以根据此进行判断是通过还是不通过；

9.红外测试
    文件：不需要下载程序，只需要人工按住光敏电阻，看看是否有红外。

10.uid烧录
    文件：程序文件：test_pushid，当点烧录uid按钮时，此时就会读取之前扫描出来的序列号，再通过序列号，查找到已准备好的文件，再把此文件下载到机器在上。


11.mac烧录
文件：读此当前的MAC地址，进行烧录到机器上，工具上的mac地址会自增1，作为下一台机器使用的。


12.复位键测试
文件：程序文件：test_recover_dev，再进行人工短按机器上的复位键1秒，看是否有效。



13,界面上的复位设备测试
文件：文件：需要下程程序test_recover_dev，机器后自动重启，此时机器恢复出厂设备；
